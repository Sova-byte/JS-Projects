# JavaScript Projects Repository

Welcome to my **JavaScript Projects Repository**!  
This repo contains **10 interactive and beginner-friendly JavaScript projects** built using **vanilla HTML, CSS, and JavaScript**. Each project is designed to practice core concepts like DOM manipulation, event handling, local storage, date/time operations, and form validation.

---

## Projects Included

| # | Project Name               | Description                                                                 | Key Features & Concepts Used                     |
|---|----------------------------|-----------------------------------------------------------------------------|--------------------------------------------------|
| 1 | **Astrology Reading**      | Enter your full name and date of birth to get a personalized astrology reading. | Form input, Date parsing, String manipulation, Conditional logic |
| 2 | **Background Color Changer** | Click on colored buttons to instantly change the page background.          | Event listeners, DOM styling, CSS classes         |
| 3 | **Clicker (Circle Burst)** | Click anywhere on the screen — a colorful growing circle appears and fades. | `mousedown`, `createElement`, `setTimeout`, CSS animations |
| 4 | **FIFA 2026 World Cup Countdown** | Live countdown timer to the FIFA World Cup 2026 opening match.            | `setInterval`, Date object, Dynamic UI updates   |
| 5 | **Digital Clock**          | Real-time digital clock with hours, minutes, and seconds.                   | `setInterval`, `toLocaleTimeString`, Formatting  |
| 6 | **Football Quiz**          | 5-question quiz about football. Score displayed at the end (out of 5).      | Arrays, Loops, Radio buttons, Score tracking     |
| 7 | **Income Tax Calculator**  | Calculate tax based on annual income using Indian tax slabs (or custom).    | Form validation, Conditional calculations, Output formatting |
| 8 | **Love Calculator**        | Enter two names to get a fun love compatibility score (0–100).              | Random generation, Input validation, Fun UI      |
| 9 | **Random Quote Generator** | Click a button to display a new inspirational quote.                        | Arrays, `Math.random`, Dynamic text update       |
| 10| **Todo List**              | Add, mark as done, and delete tasks. Persists using `localStorage`.         | CRUD, Local Storage, Event delegation            |

---

## Tech Stack

- **HTML5** – Semantic structure
- **CSS3** – Responsive design, animations, flexbox/grid
- **JavaScript (ES6+)** – Pure vanilla JS (no frameworks)

> All projects are **framework-free** and run in any modern browser.
